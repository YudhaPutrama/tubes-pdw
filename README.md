### File dan Folder
- Untuk file HTML (\*.html) masukkan dalam folder utama
- Untuk file style (\*.css) masukkan dalam folder /css
- Untuk file script (\*.js) masukkan dalam folder /js
- Untuk file gambar (\*.jpg / \*.png) dalam folder /img
